# Portf-lio-Desenvolvimento-Responsivo
Portfólio Faculdade ADS
<br><br>
<h2>Introdução</h2>

<p>A atividade atende ao objetivo da aula prática: implementar responsividade em página web para múltiplos dispositivos, utilizando Replit (template HTML/CSS/JS) e técnicas de media queries. Justificativa: em desenvolvimento web moderno, sites devem se adaptar automaticamente a smartphones, tablets e desktops para melhor UX. Objetivo geral: criar página com 6 blocos responsivos. Específicos: aplicar breakpoints 1280px/768px/480px e testar no navegador.</p><br>
<b>Materiais e Infraestrutura:</b>
<br>
    Plataforma: Replit.com (template "HTML, CSS, JS" - freeware).

    Equipamentos: Computador pessoal com acesso à internet.

    Softwares: Navegador Chrome/Firefox (DevTools para simular tamanhos de tela).

    Arquivos criados: index.html e style.css (JS não utilizado).
<br>
<b>Procedimentos Realizados:</b>
<br>
<b>Segui os passos do roteiro:</b>
<br>
    <b>Passo 1:</b> Acessei https://replit.com, criei novo Repl com template "HTML, CSS, JS".
<br>
    <b>Passo 2:</b> No index.html, estruturei página com <div class="container"> contendo 6 blocos (.bloco) com bandeiras fictícias e dados de países (França, Espanha, EUA, Itália, México, Tailândia).
<br>
    <b>Passo 3:</b> No style.css, usei CSS Grid base (1fr por bloco) e media queries para adaptação:
<br>
        ≥1280px: grid 6 colunas (desktop).

        ≥768px: grid 3 colunas (tablet).

        ≥480px: grid 2 colunas (dispositivo médio).

        <480px: grid 1 coluna (mobile).
<br>
    <p>Testes: Cliquei "Run" no Replit, redimensionei janela e usei DevTools para validar breakpoints. Responsividade confirmada em todos os cenários.</p>
